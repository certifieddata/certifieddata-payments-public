CertifiedData — Receipt Verifier

Public, independent, machine-checkable. Paste a rcpt_… ID to verify its Ed25519 signature and SHA-256 hash off-platform. 



  

    How it works
    

      Clicking Verify issues a public GET to
      /api/payments/verify/<receipt_id> on the configured
      base URL. No API key. No login. The response includes
      valid, hashValid, signatureValid,
      signingKeyId, signatureAlg.
    

    

      Change the base URL by appending ?base=http://localhost:3456
      to point at your local mock server.
