# Changelog

## 0.1.0 — 2026-04-06

Initial public release of the CertifiedData Payments public SDK and contract repository.

### Included

- OpenAPI 3.1 REST contract
- AsyncAPI event contract
- JSON Schemas for all resources (payee, payee alias, payout destination, customer, invoice, payment intent, transaction, receipt, settlement, refund, event, webhook endpoint, idempotency record, provenance link set)
- Machine-readable error catalog
- Machine-readable state machines (transaction, payment intent, settlement, refund, invoice, payout destination)
- Capabilities manifest
- Constraints manifest
- Support matrix
- TypeScript SDK (`@certifieddata/payments`)
- Python SDK skeleton (`certifieddata-payments`)
- Mock server
- JSON examples
- Test vectors (webhook signature, idempotency, provenance)
- Documentation stubs
- `api-manifest.json` discovery entrypoint
- `llms-full.txt` machine-facing repository guide
